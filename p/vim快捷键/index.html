2025-06-18

常用快捷键

快捷键 英文全称 作用描述
i Insert 进入插入模式（当前光标前）
a Append 进入插入模式（当前光标后）
o Open line below 在当前行下方插入新行并进入插入模式
O Open line above 在当前行上方插入新行并进入插入模式
ESC Escape 退出插入模式，返回普通模式
h j k l Left/Down/Up/Right 方向键（左/下/上/右）
w Word 移动到下一个单词开头
b Back 移动到上一个单词开头
e End of word 移动到当前单词末尾
0 Zero 移动到行首
$ End of line 移动到行尾
gg Go to top 移动到文件第一行
G Go to bottom 移动到文件最后一行
:n Line number 跳转到第 n 行（如 :10 跳转到第 10 行）
x Delete character 删除当前光标下的字符
dd Delete line 删除当前行
yy Yank (copy) line 复制当前行
p Paste 粘贴到光标后
P Paste before 粘贴到光标前
u Undo 撤销上一步操作
Ctrl + r Redo 重做撤销的操作
/pattern Search forward 向前搜索指定文本
?pattern Search backward 向后搜索指定文本
n Next match 跳转到下一个匹配项
N Previous match 跳转到上一个匹配项
:%s/old/new/g Substitute globally 全局替换文本（将 old 替换为 new）
:w Write 保存文件
:q Quit 退出 Vim
:wq Write and quit 保存并退出
:q! Quit without saving 强制退出（不保存修改）
v Visual mode 进入可视模式（字符选择）
V Visual line mode 进入可视模式（行选择）
Ctrl + v Visual block mode 进入可视模式（块选择）

2. 插入模式 i Insert 光标前插入
a Append 光标后插入

O Open line above 上方插入新行

Ctrl + r {reg} Insert register 插入寄存器内容
3. 删除操作 x Delete character 删除光标下字符
X Delete before 删除光标前字符
dd Delete line 删除整行
dw Delete word 删除到单词尾
d$ / D Delete to end 删除到行尾
d^ Delete to start 删除到行首
dG Delete to EOF 删除到文件尾
dgg Delete to BOF 删除到文件头
d{motion} Delete motion 按动作删除
J Join lines 合并当前行和下一行
4. 修改与替换 r{char} Replace character 替换单个字符
R Replace mode 进入替换模式
cc Change line 修改整行
cw Change word 修改单词
c$ / C Change to end 修改到行尾
c^ Change to start 修改到行首
~ Toggle case 切换大小写
gU{motion} Uppercase motion 转大写
gu{motion} Lowercase motion 转小写
g~~ Toggle case line 整行切换大小写
xp Transpose characters 交换相邻字符
>> Indent right 增加缩进
<< Indent left 减少缩进
5. 复制粘贴 yy Yank line 复制整行
yw Yank word 复制单词
y$ Yank to end 复制到行尾
y^ Yank to start 复制到行首
p Paste after 粘贴到光标后
P Paste before 粘贴到光标前
"*y Yank to system clipboard 复制到系统剪贴板
"*p Paste from system clipboard 从系统剪贴板粘贴
]p Paste with indent 自动缩进粘贴
6. 撤销与重做 u Undo 撤销操作
Ctrl + r Redo 重做操作
U Undo line 整行撤销
7. 搜索与替换 /pattern Search forward 向前搜索
?pattern Search backward 向后搜索
n Next match 下一个匹配项
N Previous match 上一个匹配项
* Search word under cursor 搜索当前单词
# Search word backward 反向搜索当前单词
:%s/old/new/g Substitute globally 全局替换
:s/old/new/g Substitute in line 当前行替换
:s/old/new/gc Substitute with confirm 带确认的替换
& Repeat last substitute 重复上次替换
8. 可视模式 v Visual character 字符选择模式
V Visual line 行选择模式
Ctrl + v Visual block 块选择模式
y Yank selection 复制选中内容
d Delete selection 删除选中内容
c Change selection 修改选中内容
> Indent right 向右缩进
< Indent left 向左缩进
~ Toggle case selection 切换选中内容大小写
o Move to other end 切换选择端点
9. 窗口管理 :sp [file] Horizontal split 水平分屏
:vsp [file] Vertical split 垂直分屏
Ctrl + w s Split window 水平分割当前窗口
Ctrl + w v Vertical split window 垂直分割当前窗口
Ctrl + w h Window left 切换到左窗口
Ctrl + w j Window down 切换到下窗口
Ctrl + w k Window up 切换到上窗口
Ctrl + w l Window right 切换到右窗口
Ctrl + w c Close window 关闭当前窗口
Ctrl + w o Only window 关闭其他窗口
Ctrl + w = Equal window size 等分窗口尺寸
Ctrl + w + Increase height 增加窗口高度
Ctrl + w - Decrease height 减少窗口高度
Ctrl + w > Increase width 增加窗口宽度
Ctrl + w < Decrease width 减少窗口宽度
Ctrl + w r Rotate windows 轮换窗口位置
Ctrl + w T Move to new tab 将窗口移到新标签页
10. 标签页 :tabe [file] Tab edit 新建标签页
gt Go to next tab 下一个标签页
gT Go to previous tab 上一个标签页
:tabm [n] Tab move 移动标签页位置
:tabc Tab close 关闭当前标签页
:tabo Tab only 关闭其他标签页
11. 寄存器与宏 "{reg}y Yank to register 复制到指定寄存器
"{reg}p Paste from register 从寄存器粘贴
:reg Show registers 显示寄存器内容
qa Record macro ‘a’ 录制宏到寄存器 a
q Stop recording 停止录制
@a Execute macro ‘a’ 执行寄存器 a 的宏
@@ Repeat last macro 重复上次宏
12. 命令模式 :w Write 保存文件
:q Quit 退出 Vim
:wq Write and quit 保存并退出
:q! Quit without saving 强制退出
:e [file] Edit file 打开文件
:saveas [file] Save as 另存为
:bn Buffer next 下一个缓冲区
:bp Buffer previous 上一个缓冲区
:bd Buffer delete 关闭缓冲区
:ls List buffers 列出缓冲区
:r [file] Read file 插入文件内容
:r !cmd Read command output 插入命令输出
:!cmd Execute shell command 执行 shell 命令
13. 高级操作 . Repeat last command 重复上一次操作
Ctrl + a Increment number 增加数字
Ctrl + x Decrement number 减少数字
gf Go to file 打开光标下文件
Ctrl + ^ Switch to alternate file 切换上一个文件
:make Run make 执行 make 命令
:cnext Next error 下一个错误
:cprev Previous error 上一个错误
K Keyword help 查看光标下单词帮助
Ctrl + ] Jump to tag 跳转到标签定义
Ctrl + t Jump back from tag 从标签返回
zz Center screen 当前行居中屏幕
zt Top screen 当前行置顶
zb Bottom screen 当前行置底
g; Last change position 跳转到上次修改位置
g, Next change position 跳转到下一个修改位置
:set wrap Enable line wrap 启用自动换行
:set nowrap Disable line wrap 禁用自动换行
14. 折叠操作 zc Close fold 关闭折叠
zo Open fold 打开折叠
za Toggle fold 切换折叠状态
zR Open all folds 打开所有折叠
zM Close all folds 关闭所有折叠
zf{motion} Create fold 创建折叠区域
zd Delete fold 删除折叠
15. 书签 m{a-z} Set mark 设置书签 (a-z 本地, A-Z 全局)
`{mark} Jump to mark 跳转到书签
'{mark} Jump to mark line start 跳转到书签行首
:marks List marks 列出所有书签
16. 特殊模式 Q Ex mode 进入 Ex 模式
gQ Ex mode with return 可返回的 Ex 模式
Ctrl + v (插入模式) Insert literal character 插入原义字符
Ctrl + k {char}{char} Digraph input 输入特殊字符
17. 帮助系统 :help Open help 打开帮助文档
:help topic Help on topic 查看特定主题帮助
Ctrl + ] (帮助) Jump to tag 跳转到帮助标签
Ctrl + t (帮助) Jump back 从帮助标签返回

总说明：

  1. Vim 有 7 种基本模式：普通模式(Normal)、插入模式(Insert)、可视模式(Visual)、命令行模式(Command-line)、Ex 模式、选择模式(Select)和替换模式(Replace)
  2. 所有快捷键默认在普通模式生效，除非特别标注
  3. 可通过 :map 查看所有自定义映射
  4. 使用 :verb map <key> 可查看特定键位的映射来源
