DPI / CPI 分辨率灵敏度，CPI 为真实值，DPI 为处理后值 400–1600 常用；高端最高 30K–45K 灵敏度，指针移动速度

🖱️ 鼠标核心参数整理

DPI / CPI（分辨率灵敏度）

  • CPI (Counts Per Inch)：传感器真实记录的每英寸数据点数量（更精确的指标）。

  • DPI (Dots Per Inch)：经过 DSP 处理后的数值，广告宣传常用。

  • 常用范围：400 – 1600（职业选手常用区间，DPI 偏移较小）。

  • 意义：决定鼠标移动 1 英寸时，屏幕上指针移动的点数。

采样率（FPS, Frame Per Second）

  • 定义：CMOS 每秒拍摄表面图像的帧数。

  • 作用：保证轨迹还原精度，避免丢帧。

  • 典型值：

    • IE3.0 时代：6000–9000 fps

    • PAW3395：最高 12000 fps

    • PAW3950：最高 20000 fps

  • 注意：采样率越高，功耗越大。

回报率 / 轮询率（Polling Rate, Hz）

  • 定义：鼠标与电脑之间每秒通讯的次数。

  • 常见值：125 Hz / 500 Hz / 1000 Hz / 2000 Hz / 4000 Hz / 8000 Hz。

  • 效果：回报率越高，指针轨迹越细腻、延迟越低。

  • 代价：高回报率对 CPU 占用更高（8kHz 下 CPU 需每 0.125ms 处理一次数据）。

最大追踪速度（IPS）

  • 单位：Inch Per Second（英寸/秒）。

  • 作用：鼠标在极高速甩动时，传感器是否还能准确捕捉。

  • 一般数值：

    • 普通传感器：200–300 IPS

    • 高端传感器（3395/3950 等）：650+ IPS（超过人手极限）。

最大加速度（G）

  • 单位：重力加速度（1g ≈ 9.8 m/s²）。

  • 作用：保证在快速变向操作时不丢帧。

  • 常见值：40g – 50g（高端传感器）。

LOD（Lift-off Distance，抬升高度）

  • 定义：鼠标离开鼠标垫到一定高度后不再识别。

  • 意义：越低越好，避免频繁抬鼠标时影响视角。

  • 高端传感器：可调节，通常 <1mm–2mm。

主控（MCU，Micro Control Unit）

  • 作用：负责数据传输与带宽控制。

  • 常见厂商：Nordic、上海博通、瑞昱。

  • 问题：带宽不足可能导致「丢帧」或 DPI 偏移。

功耗与续航

  • 高采样率 + 高回报率 → 功耗暴涨（3950 传感器鼠标续航差）。

  • 优化手段：分档模式（如「狂暴竞技模式」/「日常模式」），平衡性能与续航。

🏷️ 主流传感器

  • PAW3395：

    • 12000 fps，支持 8kHz 回报率。

    • 物美价廉，大量中高端鼠标采用。

  • PAW3950：

    • 20000 fps，支持 8kHz 回报率。

    • 更细腻，支持玻璃垫、更低 LOD。

    • 部分版本 30K / 42K DPI（42K 多为插值超频）。

  • HERO（罗技专用）：

    • 与 3950 同级，功耗极低，续航优秀。

    • 模具选择少（GPW/GPXS）。

📌 总结选购要点

  1. 普通玩家：1000 Hz 回报率 + PAW3395 足够，价格适中。

  2. 进阶/职业：PAW3950 或 HERO + 高采样率/低 LOD，更稳定。

  3. 办公/轻度游戏：125–500 Hz 回报率，省电为主。
