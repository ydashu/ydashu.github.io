SVN

https://plugins.svn.wordpress.org/ 是 WordPress 官方插件的 Subversion（SVN）仓库，用于托管所有 WordPress 插件的开源代码。要从这个地址下载插件（如前面提到的 SMTP Connector），你需要使用 SVN 客户端或通过浏览器手动下载特定文件。以下是详细步骤：

SVN 是一种版本控制系统，WordPress 使用它来管理插件代码。你需要安装 SVN 客户端（如 TortoiseSVN、命令行 SVN）并执行以下步骤：

1. 安装 SVN 客户端

  • Windows：下载并安装 TortoiseSVN，这是一个图形化 SVN 工具。
  • Mac/Linux：通常自带 SVN 命令行工具，或通过包管理器安装：
    • Mac（Homebrew）：brew install subversion
    • Ubuntu/Debian：sudo apt-get install subversion

2. 确定插件的 SVN 地址

  • WordPress SVN 仓库的结构为：
    • https://plugins.svn.wordpress.org/<插件名称>/
    • 例如，SMTP Connector 的地址是 https://plugins.svn.wordpress.org/smtp-connector/
  • 仓库内通常有以下子目录：
    • /trunk/：最新开发版本。
    • /tags/：已发布的稳定版本（如 /tags/1.0.0/）。
    • /branches/：开发分支（不常用）。

3. 使用命令行下载

  • 打开终端（Windows 可使用 CMD、PowerShell，或 TortoiseSVN 的右键菜单）。
  • 输入以下命令以下载整个插件的最新版本（trunk）： 
    1

    svn checkout https://plugins.svn.wordpress.org/smtp-connector/trunk/ smtp-connector
    • 这会在当前目录下创建一个名为 smtp-connector 的文件夹，包含最新代码。
  • 如果需要特定版本（如 1.0.0），使用 tags： 
    1

    svn checkout https://plugins.svn.wordpress.org/smtp-connector/tags/1.0.0/ smtp-connector-1.0.0

4. 使用 TortoiseSVN（Windows）

  • 右键文件夹空白处，选择 “SVN Checkout”。
  • 在 “URL of repository” 中输入插件地址（如 https://plugins.svn.wordpress.org/smtp-connector/trunk/）。
  • 选择本地保存路径，点击 “OK” 下载。
