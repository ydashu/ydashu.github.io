debian12+GNOME快捷键

2025-06-18

以下是一般意义下的快捷键，具体可以在设置-键盘-键盘快捷键中查看和设置

分类 快捷键 功能
系统与全局 Super (Win 键) 打开活动概览（显示窗口、应用和搜索）
Super + A 打开应用菜单
Super + L 锁定屏幕
Ctrl + Alt + Del 打开电源选项（关机/重启/注销）
Super + Esc 打开系统监视器（资源使用情况）
Alt + F2 打开命令输入框（输入 r 可重启 GNOME Shell）
Super + V 打开剪贴板历史（需启用扩展）
Super + S 切换输入法（多语言环境）
Super + Space 切换键盘布局（多语言环境）
Super + M 切换通知面板
窗口管理 Alt + Tab 在打开的应用间切换
Alt + `（Tab 上方键） 切换同一应用的多个窗口
Super + ↑ 最大化当前窗口
Super + ↓ 恢复/最小化当前窗口
Super + ←/→ 将窗口吸附到左/右半屏
Super + Home/End 最大化/最小化窗口（部分版本）
Alt + F4 关闭当前窗口
Super + D 显示桌面（最小化所有窗口）
Alt + F7 移动窗口（用方向键调整）
Alt + F8 调整窗口大小（用方向键）
Ctrl + Alt + Num (1-9) 调整窗口大小到屏幕的 1/4、1/2 等（需扩展）
工作区 Super + PgUp / PgDown 切换到上/下工作区
Ctrl + Alt + ↑ / ↓ 切换到上/下工作区（部分版本）
Super + Ctrl + ← / → 移动当前窗口到左/右工作区
Super + Shift + PgUp / PgDown 移动当前窗口到上/下工作区
Super + Alt + ↑ / ↓ 调整工作区数量（部分版本）
应用与启动器 Super + 数字 (1-9) 启动/切换 Dock 上的应用
Ctrl + Alt + T 打开终端（默认配置）
Super + Enter 打开默认终端（部分版本）
Super + F 打开文件管理器（Nautilus）
Super + E 打开文件管理器（部分版本）
截图与录屏 PrtSc 全屏截图
Alt + PrtSc 当前窗口截图
Shift + PrtSc 区域截图
Ctrl + Alt + Shift + R 开始/停止屏幕录制（保存到 ~/Videos）
Ctrl + Shift + Alt + R 部分版本录屏快捷键
文本编辑 Ctrl + C 复制
Ctrl + X 剪切
Ctrl + V 粘贴
Ctrl + Shift + V 粘贴为纯文本（部分应用支持）
Ctrl + Z 撤销
Ctrl + Y 重做
Ctrl + A 全选
Ctrl + F 查找
Ctrl + S 保存
Ctrl + W 关闭当前标签/窗口
终端 Ctrl + Shift + T 新建标签页
Ctrl + Shift + W 关闭标签页
Ctrl + PgUp / PgDown 切换标签页
Ctrl + Shift + C 复制选中文本
Ctrl + Shift + V 粘贴文本
Ctrl + L 清屏（类似 clear）
Ctrl + D 关闭终端（或发送 EOF）
Ctrl + Shift + N 新建终端窗口
浏览器 Ctrl + T 新建标签页
Ctrl + W 关闭当前标签页
Ctrl + Shift + T 恢复关闭的标签页
Ctrl + Tab 切换到下一个标签页
Ctrl + Shift + Tab 切换到上一个标签页
Ctrl + L 聚焦地址栏
F5 / Ctrl + R 刷新页面
Ctrl + Shift + R 强制刷新（忽略缓存）
Ctrl + H 打开历史记录
Ctrl + J 打开下载列表
