安装windows-Linux双系统

2025-04-09

在windows的基础上加装linux

ubuntu或者Linux Mint

  • 下载ubuntu 官方链接
  • 下载Linux Mint 官方链接

ventory制作可启动U盘

  • 下载ventory
  • 参考文档
  • 解压
  • 插入U盘
  • 打开解压后的文件夹，双击 Ventoy2Disk.exe
  • 设备选择：下拉菜单会显示检测到的 U盘（例如 USB: SanDisk 32GB），确认无误。
  • 选项（可选）：
    • 分区设置保留空间大小
    • 默认选择 MBR 分区方案（兼容 BIOS 和部分 UEFI）
    • 若目标设备仅支持 UEFI，可点击“选项” > “分区类型” > 选择 MRB或GPT
      • 具体参考Ventoy MBR & GPT 格式对比
    • 可勾选“安全启动支持”（Secure Boot Support），若目标设备启用此功能
  • 点击“安装”按钮：
  • 会弹出警告，提示 U盘数据将被清除，确认后继续。
  • U盘分区：MBR分区参考链接，GPT分区参考链接
    • 分区 1：Ventoy 引导分区（几 MB，FAT32 格式，含引导文件），Windows 下显示为一个小的可访问盘符（如 F:）。
    • 分区 2：数据分区（剩余空间，用于存放 ISO 文件），默认不挂载，需手动分配盘符
    • 从 1.0.14 版本开始，Ventoy支持在安装时在磁盘最后保留一部分空间。在安装时可以在 “选项配置” —> “分区设置” 中设置要保留的空间大小
  • U盘的第二个分区添加ISO文件

MBR vs GPT

  • MBR 需要 Legacy/CSM：因为 BIOS 和 CSM 只能识别 MBR 的引导代码。
  • GPT 需要 UEFI：因为 GPT 依赖 EFI 分区和 UEFI 的高级引导机制。
  • BIOS vs UEFI：BIOS 是老技术，仅支持 MBR；UEFI 是新技术，支持 GPT 和更多功能。
  • Ventoy 灵活性：根据目标设备选择 MBR 或 GPT，确保固件模式匹配即可
  • 老主板（2010 年前）多为 BIOS。
  • 新主板（2012 年后，尤其是 Intel 6 系列芯片组或 AMD AM3+ 之后）几乎都支持 UEFI。

安装流程

windows利用磁盘空间划分空间

  • WIN + X
  • 打开磁盘管理
  • 选择一个空间较多的盘，右键点击它，选择压缩
  • 压缩完了以后，底下会多出一块显示为“未分配”的空间，假设是100G

准备工作

  • 关闭BitLocker
    • bitlocker是微软出品的用于保护硬盘的加密系统，按下win+S搜索BitLocker，点击进入后将其关闭，否则它将禁止任何非windows的系统或设备对硬盘进行写入操作
  • 禁用快速启动：
    • 控制面板 > “电源选项” > “选择电源按钮的功能” > 取消勾选“启用快速启动
  • BIOS/UEFI禁用安全模式

流程

  • 重启电脑，插入 U盘。
  • 进入 BIOS/UEFI 设置（通常按 F2、Del 或 Esc）：
    • 设置 U盘为第一启动项。
  • 保存并重启，进入 Ventoy 启动菜单：
    • 显示所有 ISO 文件列表，用方向键选择，回车启动
  • 安装 Ubuntu/Linux Mint
    • 语言、键盘、网络连接等
  • 选择安装类型
    • 安装程序检测到 Windows，提示你想安装ubuntu吗？，选择手动安装，一定不要选择第二项
  • 设置挂载
    • 根/分区
      • 大小：输入 40000（即 40GB，足够系统和软件使用）。
      • 类型：选择“主分区”（Primary）。
      • 位置：选择“开头”（Beginning）。
      • 使用为：选择“Ext4 日志文件系统”（Ext4 journaling file system）。
      • 挂载点：选择 /（根目录）。
    • /home分区
      • 回到分区列表，选中剩余的未分配空间（现在应该是 60GB），点击“+”按钮。
      • 设置参数：
        • 大小：保持默认（使用剩余的 60GB）。
        • 类型：选择“主分区”（Primary）。
        • 位置：选择“开头”（Beginning）。
        • 使用为：选择“Ext4 日志文件系统”（Ext4 journaling file system）。
        • 挂载点：选择 /home。
    • 备注
      • 内存16g，并且一般情况下不会超内存，没必要设置交换空间 swap area
      • /boot分区暂时也不需要
  • 安装完成，选择“立即重启”，移除 U盘
    • 如果由于网络问题安装失败，可以离线安装，待安装完ubuntu之后再设置代理等方式安装额外的软件、包和驱动等
  • 重启后，进入 GRUB 菜单
    • 若重启未见 GRUB，进入 UEFI 设置，调整启动顺序至 Ubuntu/Linux Mint
    • 若 Windows 未显示，进 Ubuntu 运行 sudo update-grub

Linux的常用软件替代windows的常用软件

类目 Windows 软件及 Linux 替代品 说明或补充信息
视频播放 PotPlayer → MPV / VLC MPV 轻量可定制，VLC 功能全面，支持硬件加速，均无广告
办公套件 onlyoffice/LibreOffice/WPS2019 → Microsoft Office WPS Linux 版有广告，推荐 WPS 2019（较旧但无广告）
文本编辑 Notepad++ → Gedit / Nano / Vim Gedit 简单图形化，Nano 命令行轻量，Vim 强大但需学习
文件搜索 Everything → FSearch FSearch 开源快速，需从官网或 PPA 安装
浏览器 Chrome / Edge → Firefox / Chromium Firefox 注重隐私，Chromium 开源，洋葱（Tor）需单独配置
搜索引擎 Google → DuckDuckGo DuckDuckGo 隐私优先，无跟踪，需调整搜索习惯
代理工具 Clash for Windows → Clash Verge Clash Verge 开源，支持 Clash 核心，需从官网下载
下载工具 迅雷 → Motrix / qBittorrent Motrix 全能（BT+HTTP），qBittorrent 专注 BT，均无广告
